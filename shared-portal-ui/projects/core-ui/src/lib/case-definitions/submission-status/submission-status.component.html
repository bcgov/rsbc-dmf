Listed below are the various Submission Statuses.
  • Open Required – RSBC has requested a document and is waiting for submission
  • Received – RSBC has received the document and it is either under review or waiting to be reviewed
  • Rejected – Submitted document has been rejected by RSBC for various reasons, for example, document was cut off, illegible or duplicate.
  • Non-comply – Requested document was not submitted by the submittal deadline
  • Uploaded - A document has been uploaded but has not yet been reviewed
